EM2: Damage Types

Em_Monster types support damage subtypes

class Em_Zombieman : Em_Monster
{
    Default
    {
        DamageFactor "Bullet", 0.5;
        DamageFactor "Bullet.Heavy", 0.75;
        DamageFactor "Bullet.Heavy.Rifle", 0.99;

        DamageFactor "Fire", 0.1;
        DamageFactor "Fire.Plasma", 0.2;
        DamageFactor "Fire.Plasma.Railgun", 1.0; // Not valid because of 1.0
    }
}

So a projectile with damagetype "Bullet.Heavy.Minigun"

Would in this case apply DamageFactor "Bullet.Heavy", 0.75;

Death states

For DamageType "Explosive.Shell.Tank" the death state preference would be:

Death.Explosive.Shell.Tank
Death.Explosive.Shell
Death.Explosive
Death

or with XDeath

XDeath.Explosive.Shell.Tank
XDeath.Explosive.Shell
XDeath.Explosive
XDeath
Death.Explosive.Shell.Tank
Death.Explosive.Shell
Death.Explosive
Death

Which is different behavior than regular ZDoom where Death.Something without XDeath would take precedence
