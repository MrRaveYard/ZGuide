Introduction

Thereâ€™s absolutely no need to use #import or #include to use these scripts.

There are various utility scripts for mappers to utilize from the global script library.

All that needs to be done is calling the proper named script with correct parameters.

Scripts

// Cutscene
script "Em_Cutscene_Start" (void); // tells EM_ACS that map specified cutscene is playing. If cutscene is skipped, it'll call "Em_Skip_Cutscene"
script "Em_Cutscene_Skipper" (void); // deprecated: use "Em_Cutscene_Start" instead
script "Em_Cutscene_IsActive" (void); // returns true if cutscene skip option is shown
script "Em_Cutscene_End" (void); // tells EM_ACS that cutscene is no longer playing.

// Misc
script "em_kill_me" (int teleport_spot); // Kills activator of this script. The activator is silently teleported to the teleport_spot (if teleport_spot isn't 0)

// Music library
script "em_music_fadein" (fixed seconds);
script "em_music_fadeout" (fixed seconds);

Callback scripts

These scripts are meant to be overriden by the map scripts

// Starting Items
script "em_starting_loadout" (void); // called when player is starting new game in this map. Implement this to give players/developers appropriate starting items in this map.

// Cutscene
script "Em_Skip_Cutscene" (void); // called when player skips cutscene
